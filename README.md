# ZombieShooter
Download the game [here](https://github.com/RipperOnik/ZombieShooter/releases)
## Controls
- WASD - for moving
- LEFT and RIGHT Arrow keys - for aiming/turning 
- SPACE - for shooting

## Win condition
Kill all zombies, and don't get grabbed by them
