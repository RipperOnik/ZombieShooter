# ZombieShooter
Download the game [here](https://github.com/RipperOnik/ZombieShooter/releases/tag/v1.0.0)
## Controls
- WASD - for moving
- LEFT and RIGHT Arrow keys - for aiming/turning 
- SPACE - for shooting

## Win condition
Kill all zombies

## Lose condition
Get close to a zombie
